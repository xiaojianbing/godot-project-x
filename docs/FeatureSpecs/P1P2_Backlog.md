## 🟡 P1 - 第二支柱：世界观与叙事骨架

战斗系统稳定后开始搭建，为战斗提供**情感驱动力**和**节奏变化**。

| Feature        | 简述                                            | 竞品参考                                     | 初步验收标准                               |
| -------------- | ----------------------------------------------- | -------------------------------------------- | ------------------------------------------ |
| 关卡/区域设计  | 差异化的生态区域                                | 《空洞骑士》区域转换；《Dead Cells》程序生成 | 至少 3 个视觉和玩法差异明显的区域          |
| Boss 战        | 阶段制 Boss 设计                                | 《空洞骑士》Boss；《只狼》Boss 多阶段        | 每个 Boss 至少 2 个阶段，各有独特机制      |
| 叙事系统       | 对话/文本/过场                                  | 《空洞骑士》碎片化叙事；《Ori》环境叙事      | 不打断战斗节奏的叙事方式                   |
| 能力解锁       | Metroidvania 式能力获取（二段跳、游泳、钩索等） | 《空洞骑士》暗影冲刺；《蔚蓝》新冲刺机制     | 新能力同时拓展**探索范围**和**战斗招式库** |
| 敌人种类丰富化 | 多种行为模式的敌人                              | 《忍龙》敌人差异化；《鬼泣》敌人组合         | 至少 5 种有明显行为差异的敌人类型          |
